# ZKA C++

ZKA's Toolchain sources.

## Installing:

Start by cloning the repository.

```
git clone git@bitbucket.org:mahrouss/sdk-dll.git
```

## Running makefiles:

And then run the makefile.

```
make all
```

You can now use the programs.
